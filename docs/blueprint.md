# **App Name**: GreenScape Nursery

## Core Features:

- Product Catalog & Detail View: Browse, filter, sort, and search for plants. View detailed product pages with images, descriptions, pricing, and care guides. Add products directly to the shopping cart.
- Shopping Cart & Simplified Checkout: Manage items in the shopping cart (add, remove, adjust quantity). Proceed through a simplified checkout flow with address input and mock payment method selection.
- User Account & Order History: Allow user registration and secure login. Users can view and edit their basic profile information and access a history of their past orders with status updates.
- Affiliate Program Portal: Enable users to apply to become affiliates. Once approved by an administrator, affiliates can generate unique referral links and access a dashboard displaying basic earnings and referral statistics.
- Admin Product & Order Management: Provide an administrative interface to manage product listings (add, edit, delete plant details and placeholder images). View and update the status of customer orders.
- AI Product Description Tool: In the admin product management interface, offer an AI tool to suggest engaging, SEO-optimized product descriptions for new or existing plants based on their name and category.

## Style Guidelines:

- Primary calls to action and prominent elements use a professional, deep forest green (#1B5E20) to evoke nature and reliability, suitable for a light background.
- An accent green (#2E7D32) is used for secondary elements and to add depth to the green palette.
- Backgrounds primarily feature a clean, almost neutral white for cards and key content areas, complemented by a very subtle light green (#F1F8E9) for sections to reinforce the natural theme and a neutral background (#FAFAF7).
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly legible appearance, fitting for e-commerce content. Note: currently only Google Fonts are supported.
- Employ a consistent set of clean, modern, outline-style icons throughout the application, relevant to plant themes and e-commerce functionalities, maintaining a sense of professionalism.
- Emphasize a mobile-first, responsive design using card-based layouts. Features clean white cards with soft shadows and rounded corners (8px–16px) for a modern, inviting feel.
- Incorporate subtle, elegant animations such as smooth page transitions, gentle card hover lift effects, button ripple animations, and skeleton loaders for a polished user experience. Role-based UI animations facilitate seamless transitions for affiliate program changes.
